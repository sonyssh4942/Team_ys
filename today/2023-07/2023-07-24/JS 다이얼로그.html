JS 다이얼로그

사용자에게 메시지를 출력하거나, 입력ㅇ르 받을 수 있는 3가지 다이얼로그가 있다.

1. 프롬프트 다이얼로그, prompt("메세지", "디폴트 입력값")

2. 확인 다이얼로그, confirm("메세지")

3. 프롬프트 다이얼로그, alert("메세지")

1. 프롬프트 다이얼로그 / prompt("메세지", "디폴트 입력값")

prompt() 함수는 다이얼로그를 출력하고 사용자로부터 문자열을 입력받아 리턴한다.

'메세지' 와 '디포트 입력값' 을 입력하지만 '디폴트 입력값'은 생략이 가능하다.

아무값도 입력되지 않았으면 ""를 취소나 강제로 다이얼로그를 닫은 경우 null을 리턴한다.



2. 확인 다이얼로그 / confirm("메세지")

'메세지'와 확인/취소 버튼을 가진 다이얼로그를 출력한다.

사용자가 확인 버튼을 누르면 true 값을, 취소 버튼이나 닫으면 false를 리턴한다.



3. 경고 다이얼로그 / alret("메세지")

다이얼로그를 출력하여 단순히 메세지를 전달한다.
