let의 특징

  • var는 동일한 변수를 재선언할 수 있지만, let으로는 동일한 변수를 재선언 할 수 없다.
  • let은 변수의 사용범위를 블록 내로제한한다.

let으로는 동일한 변수를 재선언 할 수 없다.

        var x = 1;
        var x = 2;  // 정상. 기존 변수 x 제거
                    // 새로운 번수 x 생성
    
        let x = 1;
        let x = 2;  // 오류. 변수 x에대한 재선언 불가
var를 이용하면 이미 선언된 변수를 폐기하고 재선언할 수 있지만, 실수해도 오류없이 재선언하기 때문에
혼동되어 오류가 발생해도 실수를 발견하기 어렵다.

let은 변수 사용 범위를 블록 내로 제한

let으로 선언도니 변수는 블록 내에서만 사용하도록 제한한다.
블록은 {}으로 묶은 영역이다. 

        if(a == b){
            let x= 10;  // x는 if블록에서만 사용
        }
        x++;            // 오류. 사용할 수 없음
