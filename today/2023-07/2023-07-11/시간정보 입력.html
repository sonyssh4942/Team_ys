input 태그를 이용해 시간을 입력받을 수 있는 폼 요소를 만들 수 있다.
value 속성에는 날짜나 신가의 초깃값을 지정한다.

input 태그의 시간정보 type

  • month
    • : 년 / 월 - 2023-07
  • week
    • : 년 / 몇 번째 주 - 2023-W03
  • date
    • : 년 / 월 / 일 - 2023-07-11
  • time
    • : 시 / 분 - 16:20
  • datetime-local
    • : (해당 나라의 시간)? 년 / 월 / 일 / 오전,오후 / 시 / 분 / 초 / 100분의 1초 - 2023-07-11T30:10:32
