형식을 가진 텍스트 입력

input 태그의 type에는 이메일 주소, URL 전화번호와 같이
특정 형식에 맞게 입력되었는지 검사하는 폼 요소들을 제공한다.

1. 이메일주소 / input type="email"


type="text"와 모양은 같지만 이메일주소의 형식만 입력할 수 있다.
submit 버튼이 클리되면 웹 서버로 전송하기전에 형식 검사가 수행된다.
올바른 형식이 아니라면 오류메세지를 출력하고 전송하지 않는다.

2. URL 주소 / input type="url"


URL만 입력받는 태그로서 'http://www.naver.com'와 같이 W3C의 규정에 맞도록 입력되어야 한다.
웹 서버로 전송될 때 유효성이 자동으로 검사된다.
사용자의 바른 입력을 유도하기 위해 placeholder 속성 사용을 권장한다.

3. 전화번호 / input type="tel"


전화번호의 형식이 워낙 다양하므로, 이 태그는 유효성 검사를 하지 않는다.
사용자의 바른 입력을 위해 placeholder 속성 사용을 권장한다.

4. 검색어 입력 / input type="search"


검색어 입력 창을 만들 수 있다.

폼 요소들의 그룹핑 / fieldset

fieldset 태그를 이용하여 폼 요소들을 그룹으로 묶을 수 있다.
그룹의 제목은 legend 태그로 표현한다. 브라우저는 그룹을 외곽선 박스와 제목을 출력하여 표현한다.
