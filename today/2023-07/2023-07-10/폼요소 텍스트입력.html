텍스트 입력

- form 태그 내에서 사용자에게 텍스트 정보를 입력받기 위해 input, textarea 태그를 사용할 수 있다.
주소나 이름 등 한 줄 텍스트를 입력받는 경우 input태그로 만들 수 있다.

input 태그
----- [type= "text" , value= "초기출력 문자"] 로 작성

----- [type= "password"] 로 작성, password는 입력하는 문자가 *로 표시된다.

-------------------------------------------------------------------------------------------------------------

여러줄의 텍스트를 입력받을 때는 textarea 태그로 만들 수 있다.

----- [cols="20" rows="5"] 로 작성

input / type= text,password

  • name : 요소이름
  • maxlength : 최대 문자 개수
  • size : 입력창의 크기
  • value : 초기 텍스트

textarea

  • cols : 열 개수
  • rows : 행 개수
  • name : 요소 이름
  • wrap : 자동 줄바꿈 처리 지정(off hard soft)
