선택형 입력

input 태그의 checkbox / radio type을 이용하여 목록에서 사용자가 선택하는 방식의 입력요소를 만들 수 있다.
checked 속성을 사용하여 사용자가 선택하기 전에 선택상태로 만들 수 있다.

체크박스

체크박스는 선택/해제 중 하나를 선택하는 폼 요소이다.

Q.오늘 저녁 메뉴는?
한식 ---- checked 속성입력
중식
일식
양식

라디오 버튼

라디오 버튼은 name속성 값이 같은 버튼들이 하나의 그룹을 형성하여 하나만 선택 하는 폼 요소이다.

Q.오늘 저녁 메뉴는?
짜장면
짬뽕
볶음밥
비빔밥 ------ name속성 값이 달라 중복 선택할 수 있다.

콤보박스

select / option 태그를 이용해 선택지가 있는 콤보박스를 만들 수 있다.
option 태그의 value 속성 값이 submit 됐을때 전송되는 값이다.
option 태그의 selected 속성으로 선택상태로 만들 수 있다.

select / option


datalist / option

나라 : -----[input type="text" list="contry"]

*주의사항
datalist 태그는 옵션 리스트에서 선택하거나 직접입력하지만
select 태그는 선택만 가능하다
