텍스트 / 이미지 버튼 만들기

- 버튼은 input 태그나 button 태그로 만들 수 있다.

- type 속성으로 버튼의 기능을 선택 할 수 있다. /

버튼의 종류

1. 단순 버튼(type="button")
: 아무 기능이 없는 버튼으로 자바스크립트 코드에서 이용 된다.


2. submit 버튼(type="submit")
: 클릭하면 브라우저가 폼 데이터를 웹 서버로 전송한다.


3. reset 버튼(type="reset")
: 폼에 입력된 내용을 모두 지우고 초기화하는 버튼을 만든다.


4. 이미지 버튼(type="image")
: 이미지 버튼을 만드는 것으로 두가지 방법이 있다.
: 이미지 버튼들은 아무 기능이 없으며 JS 에서 활용한다.
: 버튼 태그를 이용하여 submit 이나 reset 기능을 가진 이미지 태그도 만들 수 있다.
4-1 [input type="image" src="이미지 경로"]
4-2 [button (img src="이미지 경로") /button]
