웹 폼

웹 페이지를 통해 사용자 입력을 받는 폼을 웹폼 혹은 폼이라고 한다.
input, textarea, select 등 폼을 만들기 위한 다양한 태그들을 제공하며 이것들을 '폼태그' 라고 한다.

form

- 웹 서버에 정보를 제출하기 위한 양식범위를 지정
- form 이 다른 form을 자식으로 둘 수 없음

form 태그의 속성

  • action : 전송한 정보를 처리할 웹페이지의 URL

  • autocomplete : 사용자가 이전에 입력한 값으로 자동완성 할것인지 여부확인 (자동완성기능)

  • method : 폼 데이터를 웹 서버에 전송하는 방식
    • get : 정보 공개 방식
    • post : 정보 비공개 방식

  • name : 고유한 양식의 이름, JS에서 사용

  • novalidate : 서버로 전송 시 양식 데이터의 유효성을 검사하지 않도록 지정

  • target : 서버로 전송 후 응답받을 방식을 지정(출력할 윈도우 지정)
    • self : 현재 윈도우
    • blank : 새 윈도우
