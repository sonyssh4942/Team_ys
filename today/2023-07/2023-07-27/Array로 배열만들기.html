Array로 배열 만들기

코어 객체인 Array를 이용하여 배열을 생성할 수 있으며, [ ] 로 만든 배열과 똑같이 다룬다.

초기 값을 가진 배열 생성

다음과 같이 new를 이용하여 Array 객체를 생성한다.

let week = new Array("월", "화",..."일");

new Array()로 생성한 배열도 0부터 시작하는 인덱스와 [ ] 를 이용하여 다음과 같이 원소를 접근한다.



초기화되지 않은 배열 생성

배열 객체를 일단 생성한 후 나중에 원소에 값을 저장할 수 있다.
