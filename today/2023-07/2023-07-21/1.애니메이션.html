동적변화

자바스크립트를 사용하지 않고 CSS 만으로 HTML 태그의 모양에 동적인 변화를 만들 수 있다.
CSS로 만들 수 있는 동적이 변화는 다음 3가지 이다.
  • 애니메이션
  • 전환 (transition)
  • 변환 (transform)

애니메이션

HTML 태그의 모양 변화를 시간 단위로 설정하여 쉽게 애니메이션 효과를 만들 수 있다.
애니메이션 효과를 만들기 위해서는 다음 2가지 작업이 필요하다.

웹에서 애니메이션은 JS 코드로 작성했지만 CSS 애니메이션이 만들기 쉽고,
브라우저가 최적화하여 직접 지원하므로 JS보다 속도나 렌더링이 부드럽다는 장점이 있다.

1. 시간별 애니메이션 장면 작성

@keyframes를 이용하여 시간 별로 HTML 태그가 변화하는 장면들을 작성한다. 

        @keyframes name{
            0% { color : red; }     
            50% { color : blue; }
            100% { color : green; }
        }

2. 애니메이션 스타일 시트 작성

애니메이션을 적용할 HTML 태그에 애니메이션 스타일 시트를 작성한다.
  • animation-name : 애니메이션 이름
  • animation-duration : 시간
  • animation-iteration-count : 애니메이션 반복 횟수
