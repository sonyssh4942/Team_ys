콘텐츠 자르기 overflow

overflow 는 HTML 콘텐츠가 width와 height 프로퍼티에 주어진 크기를 넘어가는 경우 자를 것인지 말 것인지를 지정한다.

블록태그에만 적용되다. overflow 프로퍼티를 적용할려면 width / height 크기설정이 있어야 한다.


overflow : visible / hidden / scroll / auto

  • visible : 콘텐츠가 잘리지 않고 태그 영역을 넘어 출력 (디폴트)
  • hidden : 넘어가는 것은 잘려서 보이지 않음 (스크롤바 없음)
  • scroll : 스크롤바를 항상 부착하여 출력
  • auto : 콘텐츠가 박스의 높이를 넘어가면 스크롤바 자동 생성
