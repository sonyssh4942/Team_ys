조건문

조건문은 조건을 검사하여 참인지 거짓인지에 따라 서로 다른 작업을 실행하는 JS문장이다.

조건문은 if-else 와 switch 두종류가 있다.

if 문

조건식이 참이면 실행문을 실행한 후 if문을 벗어나고, 거짓이면 바로 벗어난다.

if( 조건식 ){
...실행문... //조건이 참일경우
} 

    var x = 15;
    var y = 10;

    if( x > y){
        document.write("x가 큽니다")
    }

if-else 문

if-else문을 이용하면 다음과 같이 조건식이 참인 경우와 거짓인 경우데 다른 코드를 실행할 수 있다.

if( 조건식 ){
...실행문... //조건이 참일경우
}else{
...실행문2... //조건이 거짓일 경우
} 

        var x = 10;
        var y = 15;
    
        if( x > y){
            document.write("x가 큽니다")
        }else{
            document.write("y가 큽니다")
        }

다중 if-else 문

if-else문을 연속 작성하여 각 조건에 맞는 코드를 실행할 수 있다.

if( 조건식 ){
...실행문... //조건이 참일경우
}else if(조건식2) {
...실행문2... //조건2가 참일 경우
}else{
...실행문3... //조건1, 2가 거짓일 경우
} 

        var x = 10;
        var y = 10;
    
        if( x > y){
            document.write("x가 큽니다")
        }else if( x < y){
            document.write("y가 큽니다")
        }else{
            document.write("x와 y가 같습니다.")
        }
