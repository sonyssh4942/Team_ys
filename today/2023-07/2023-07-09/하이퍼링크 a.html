하이퍼링크

하이퍼링크는 a 태그를 사용하며 href 속성을 이용해 주소를 입력한다.

a 태그의 속성들
  • href : 이동할 HTML 페이지의 URL 혹은 HTML 페이지 내 앵커 이름
  • target : 링크에 연결된 HTML 페이지가 출력될 윈도우 이름 지정
  • download : 링크가 클릭되면 href에 지정된 파일이 다운로드됨

하이퍼 링크 target 속성

target 속성에는 4가지의 선택지가 있다.
  • _blank : 새 윈도우
  • _self : 현재 윈도우
  • _parent : 부모 윈도우
  • _top : 브라우저 윈도우
네이버 하이퍼링크(_blank 새 윈도우)
네이버 하이퍼링크(_self 현재 윈도우)
네이버 하이퍼링크(_parent 부모 윈도우)
네이버 하이퍼링크(_top 브라우저 윈도우)

id 속성으로 앵커 만들기

긴 내용을 가진 웹페이지를 읽을때 사용자는 스크린을 오르락내리락하며 읽어야 한다.
그래서 문서 내에서 특정 위치로 이동하는 링크를 만들어야 한다.
이러한 링크를 앵커라고 한다.
특정 부분에 id 속성으로 고유 이름을 지정하고 a 태그의 href 속성에 #앵커이름(id값)을 입력하여 만들 수 있다.
