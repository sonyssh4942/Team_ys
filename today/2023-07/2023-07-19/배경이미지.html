배경 이미지

html 태그에 배경색을 칠하거나 배경 이미지를 출력하는 방법을 알아보자

배경에 색을 칠할까, 이미지로 그릴까?

배경 색과,배경 이미지 둘이 동시에 지정되면 배경색을 먼저 칠하고 배경 이미지를 배치한다.



배경 이미지의 위치

박스 내 배경 이미지의 시작 위치를 지정한다.
[ 위치1 ][ 위치2 ] 형식으로 지정할 수 있다.

background-position: left center;


배경 이미지 반복출력

배경 이미지의 반복 출력하는 방식을 지정한다.

  • repeat : 반복 (디폴트)
  • no-repeat : 반복안함
  • repeat-x : x방향만 반복
  • repeat-y : y방향만 반복
