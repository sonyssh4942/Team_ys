텍스트 폰트

1. font-style : 글자 기울기 지정


  • normal : 스타일 없음
  • italic : 이텔릭체 (활자) (기울기)
  • oblique : 기울어진 글자 (잘안씀)
normal
italic
oblique

2. font-weight : 글자 두께 지정


  • normal : 기본글자, 두께 400과 동일
  • bold : 두껍게, 두께 700과 동일
  • bolder : 부모요소보다 두껍게 (bold 보다 두꺼운게 아님)
  • lighter : 부모요소보다 얇게
  • 숫자 : 100 ~ 900 까지의 100단위 숫자
normal
bold
bolder
lighter

3. font-size : 글자 크기 지정

  • 단위 : px / em / cm / 등 단위 지정 (기본값 : 16px)
  • %
  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large
  • smaller : 부모요소보다 작게
  • larger : 부모요소보다 크게

4. font-family : 글꼴 지정 (운영체제에 따라 다름)

구체적인 폰트 이름을 지정한다. 하지만 폰트가 브라우저에서 지원되지 않을 경우를 대비하여 여러개의 폰트를 지정한다.
후보에서 선택을 못 할 경우, 지정한 글꼴계열에서 비슷한것을 찾아 선택한다.

font-family : 글꼴후보1, 글꼴후보2, 글꼴계열

    글꼴 계열들
  • serif : 바탕체 계열
  • sans-serif : 고딕체 계열
  • monospace : 고정너비(가로폭이 동등한) 글꼴계열
  • cursive : 필기체 계열
  • fantasy : 장식(재미있는 문자표현을 포함하는) 글꼴계열
